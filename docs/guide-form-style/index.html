定制表单样式

我们可以直接在src文件下建一个custom.css文件，在这个文件中添加一些自定义样式，然后在src/index.ts文件中引入样式文件。

src/custom.css
/**
* 定制列表表头单元格的背景色
*/
.sinoform-app-inited .sinoform-table .sinoform-table-head-cell {
  background: rgb(46 83 112);
}
warning

我们在做样式定制时，一定要注意样式优先级的问题。因此在编写样式代码时，建议始终添加.sinoform-app-inited。编写完成查看效果时，如果样式没起作用，可以在控制台 查看与哪些表达式下的样式规则冲突，从而调整我们的 css 表达式。

src/index.ts
/**
 * 引入自定义样式文件
 */
import "./custom.css";

更多元素的class名称可参考表单应用支持定制样式的元素名称。
