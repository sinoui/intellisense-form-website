## 2. 平滑升级

可以从任何版本平滑升级到 v1.13.9。

## 3. sinomatrix 依赖变更

因 sinomatrix 缺陷影响了智能表单之前版本（v1.12.0 版本之前的版本）提供的功能，sinomatrix 已经修复，所以需要同步升级 sinomatrix 到 [1.6.37](http://47.93.34.153:10080/sino-matrix/sino-matrix/blob/master/CHANGELOG.md) 。（注意：访问此地址时可能会出现 ERR_UNSAFE_PORT 的情况，请参看[这篇帖子](https://blog.csdn.net/kjcxmx/article/details/118122483)配置一下浏览器。）

## 4. 主要变更

- fix: 修复导出 excel 时，多个附件没有换行展示
- fix: 修复子表数据为空时，整条数据不展示的 bug
- improve: 支持以单表单首页的方式显示应用中的表单（在单表单首页链接中添加 `forceUseFormPage` 请求参数即可，如 http://localhost:8080/intellisense-form/form-app/form-123/home/todo-list?forceUseFormPage；特别注意的如果希望推送的待办页打开的单表单首页，此待办列表页需要添加上 `forceUseFormPage` 参数）
- fix: 修复子表单子项的 label 因为标题过长显示不全的 bug
