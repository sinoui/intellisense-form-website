插值表达式

使用场景​

  • 单行文本、多行文本生成默认值
  • 说明文字生成默认的富文本内容

语法规则​

花括号包裹要动态解析的表达式字段

{ 表达式字段 }

支持的表达式字段​

表达式字段取值说明
拟稿人当前登录人姓名
填报人当前登录人姓名
创建人当前登录人姓名
流程发起人当前登录人姓名
当前登录人当前登录人姓名
办理人当前登录人姓名
发送人当前登录人姓名
拟稿人部门当前登录人部门名称
填报人部门当前登录人部门名称
流程发起人部门当前登录人部门名称
创建部门当前登录人部门名称
创建人部门当前登录人部门名称
完整部门当前登录人部门名称
创建人完整部门当前登录人完整部门路径，以/拼接
部门简称当前登录人部门名称简称
创建人部门简称当前登录人部门名称简称
创建时间当前系统时间，格式YYYY-MM-DD
拟稿时间当前系统时间，格式YYYY-MM-DD
年份当前系统时间的年份
季度当前系统时间所在的季度。第 1 季度、第 2 季度、第 3 季度、第 4 季度
月份当前系统时间所在的月份

特殊说明​

弹窗子表单支持新建的时候带入主表单表单项的默认值，格式: {主表单表单项名称}
