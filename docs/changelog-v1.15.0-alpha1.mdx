## 2. 平滑升级

可以从任何版本平滑升级到 v1.15.0-alpha.1。

## 3. sinomatrix 依赖变更

因 sinomatrix 缺陷影响了智能表单之前版本（v1.12.0 版本之前的版本）提供的功能，sinomatrix 已经修复，所以需要同步升级 sinomatrix 到 [1.6.37](http://47.93.34.153:10080/sino-matrix/sino-matrix/blob/master/CHANGELOG.md) 。（注意：访问此地址时可能会出现 ERR_UNSAFE_PORT 的情况，请参看[这篇帖子](https://blog.csdn.net/kjcxmx/article/details/118122483)配置一下浏览器。）

## 4. 主要变更

- feat: 所有可定制列表的【查询项定制】、【列表项定制】增加上移、下移的排序功能 [I4YF22](https://gitee.com/sinoui/sino-intellisense-form-test/issues/I4YF22)
- feat: 支持列表的数据列渲染定制
- feat: 分组标签添加 className [I52BDB](https://gitee.com/sinoui/sino-intellisense-form-test/issues/I52BDB)
- feat: 分组标签支持添加扩展的 className [I52BDB](https://gitee.com/sinoui/sino-intellisense-form-test/issues/I52BDB)
- feat: 支持无流程表单详情页按钮定制
- fix: 修复插值公式的“月份”获取错误 [I52B6P](https://gitee.com/sinoui/sino-intellisense-form-test/issues/I52B6P)
- fix: 修复带有扩展列表配置时发布失败的缺陷 [I52FQ7](https://gitee.com/sinoui/sino-intellisense-form-test/issues/I52FQ7)
- feat: 意见表单项可以配置同时存在同意选项和意见输入框 [I51JGG](https://gitee.com/sinoui/sino-intellisense-form-test/issues/I51JGG)
- feat: 保存、发送时，显示详细的表单校验错误 [I51JI9](https://gitee.com/sinoui/sino-intellisense-form-test/issues/I51JI9)
