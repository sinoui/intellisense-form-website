## 2. 平滑升级

可以从任何版本平滑升级到 v1.18.11。

## 3. sinomatrix 依赖变更

sinomatrix 针对当前版本的智能表单修复了一些缺陷，需要同步升级 sinomatrix 到
[1.6.49](http://47.93.34.153:10080/sino-matrix/sino-matrix/blob/master/CHANGELOG.md)
。（注意：访问此地址时可能会出现 ERR_UNSAFE_PORT 的情况，请参
看[这篇帖子](https://blog.csdn.net/kjcxmx/article/details/118122483)配置一下浏览
器。）

## 4. 主要变更

- feat: @sinoform/app-setting 新增 `listen` 方法，监听内部表单事件。见
  [监听表单事件教程](../config-guide-form-events)
- feat: 自定义选项支持 `disabled` 属性，用于禁用指定的选项
- feat:
  [@sinoform/form-data-api](https://www.npmjs.com/package/@sinoform/form-data-api)
  模块改进增删改方法的易用性
  - 第一个参数可以传递表单设计 id
  - 提供不区分表单类型（无流程表单和流程表单）的`save`、`update`、`remove`方法
