## 2. 平滑升级

可以从任何版本平滑升级到 v1.18.0。

## 3. sinomatrix 依赖变更

sinomatrix 针对当前版本的智能表单修复了一些缺陷，需要同步升级 sinomatrix 到 [1.6.49](http://47.93.34.153:10080/sino-matrix/sino-matrix/blob/master/CHANGELOG.md) 。（注意：访问此地址时可能会出现 ERR_UNSAFE_PORT 的情况，请参看[这篇帖子](https://blog.csdn.net/kjcxmx/article/details/118122483)配置一下浏览器。）

## 4. 主要变更

- feat: 支持分阅功能
- feat: 添加按钮名称配置
- feat: 添加批量生成 PDF 功能
- feat: 仪表盘支持中国地图
- feat: 表单字段支持浮动提示语
- feat: 默认值公式新增除法运算
- feat: 子表单支持设置默认值和值关联公式
- feat: 列表页支持定制编辑、删除按钮的显示
- feat: 自动关联表单项支持设置子表单项的值关联
- feat: 下拉选择复选时，支持通过配置控制是否显示复选框
- feat: 自动关联表单项已经存在值在首次点击的时候触发查询动作
- feat: 子表单中的下拉选择选项类型支持设置`其他表单项的级联选项`
- feat: 下拉选择、单选按钮组、复选按钮组支持将字典项的标识映射到其他字段
- feat: 下拉选择框的“其他表单项的级联选项”选项类型支持与多选下拉框和多选按钮组关联
- feat: 单选按钮组、复选按钮组、下拉选择的选项类型为数据字典时，支持设置数据字典默认值
- feat: 下拉选择框选择了“其他表单项的级联选项”选项类型，可以设定“限定一个分组只能选择一项”和“选中项按照分组顺序排序”
- improve: 优化判断容器组件的逻辑
- improve: 添加自定义导出按钮上传模板文件要求并输出导出失败时的错误信息
- fix: 修复下拉框的 `ids` 字段值顺序与 `titles` 字段值顺序不一致的缺陷
- fix: 修复行布局、列布局等容器表单项的子表单项默认值不生效的问题
- fix: 修复金额输入框打印输出只有数值的问题，带上币种和数值单位
- fix: 修复必填表单项的星号`*`位置不正确的 bug
- fix: 调整表单系统集成当前表单的判断条件，修复已发布的表单出现两条资源位置的问题
- fix: 修复表单发布时拖拽到二级导航下资源的 superId 不正确的问题
