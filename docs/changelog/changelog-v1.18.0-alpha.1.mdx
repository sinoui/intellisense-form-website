## 2. 平滑升级

可以从任何版本平滑升级到 v1.18.0-alpha.1。

## 3. sinomatrix 依赖变更

sinomatrix 针对当前版本的智能表单修复了一些缺陷，需要同步升级 sinomatrix 到 [1.6.49](http://47.93.34.153:10080/sino-matrix/sino-matrix/blob/master/CHANGELOG.md) 。（注意：访问此地址时可能会出现 ERR_UNSAFE_PORT 的情况，请参看[这篇帖子](https://blog.csdn.net/kjcxmx/article/details/118122483)配置一下浏览器。）

## 4. 主要变更

- chore: 单选按钮组、复选按钮组、下拉选择的选项类型为数据字典时，支持设置数据字典默认值
- feat: 添加按钮名称配置
- feat: 下拉选择框的“其他表单项的级联选项”选项类型支持与多选下拉框和多选按钮组关联
- feat: 下拉选择框选择了“其他表单项的级联选项”选项类型，可以设定“限定一个分组只能选择一项”和“选中项按照分组顺序排序”
- fix: 修复下拉框的 `ids` 字段值顺序与 `titles` 字段值顺序不一致的缺陷 (#15)
- fix: 修复行布局、列布局等容器表单项的子表单项默认值不生效的问题
