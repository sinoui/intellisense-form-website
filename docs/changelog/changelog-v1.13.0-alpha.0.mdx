## 1. 发布情况

## 2. 平滑升级

可以从任何版本平滑升级到 v1.13.0-alpha.0。

## 3. sinomatrix 依赖变更

因 sinomatrix 缺陷影响了智能表单之前版本（v1.12.0 版本之前的版本）提供的功能，sinomatrix 已经修复，所以需要同步升级 sinomatrix 到 [1.6.31](http://47.93.34.153:10080/sino-matrix/sino-matrix/blob/master/CHANGELOG.md) 。（注意：访问此地址时可能会出现 ERR_UNSAFE_PORT 的情况，请参看[这篇帖子](https://blog.csdn.net/kjcxmx/article/details/118122483)配置一下浏览器。）

## 4. 主要特性

- improve(bff): 拟稿人从查询列表页打开的已撤办的表单应可以修改
- feat: 添加打印按钮权限控制
- feat: 添加配置 draft-print-button，控制起草页面是否显示打印按钮
- fix: 表单设置流程使用范围为父部门时，子部门成员在移动端查看不到该表单
- fix: 修复无流程表单无打印按钮权限的缺陷
- improve: 改进`isIn`方法使其支持带备注选项的简单匹配
- fix: 缩减查询值为 200 最大长度
- feat: 支持自定义流程待办标题
- fix: 修复旧版页面路由跳转错误
- improve: 优化在线自定义打印模板
- feat: 打印按钮可配置
- feat: 文本字段默认值中，新增“创建人完整部门”的选项，要求：部门/处室/科室
- improve: 附件上传支持自定义上传类型
- improve: 附件上传添加选完文件之后的校验

draft-print-button 配置说明

```yaml title="application.yml"
sinoform:
  # 起草页是否显示打印按钮，默认true-显示
  draft-print-button: false
```
